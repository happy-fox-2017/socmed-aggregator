# twatt

twatt is app for us to Access twitter using postman apps 
1. post something to twitter
```
localhost:3000/api/twitt
```

2. search something on twitter
```
localhost:3000/api/twitt/:search
```

3. get timeline our user
```
localhost:3000/api/twitt
```

### how to run

```
npm install
nodemon app.js
```
